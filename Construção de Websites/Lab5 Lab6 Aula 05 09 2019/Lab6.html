LAB6 trabalho em HTML

Páginas HTML podem ser criadas em qualquer editor ou processador de texto. A única exigência é que o programa seja capaz de salvar o documento no formato texto simples, ou seja, sem nenhuma formatação esoecial, como fontes ou alinhamento de parágrafos

Mas isso não significa que você não pode usar fontes ou outras formatações de texto no documento HTML. Esses recursos são inseridos no documento de forma diferente, usando marcações chamadas tags.

As tags são usadas também para inserir imagens (acima) ou para criar hyperlinks como este, que permite acessar o site de Gifs: GIPHY.
